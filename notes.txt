Shortcuts:
-----------

control + }  -> comentar/descomentar lineas

shift + alt + a -> comentar/descomentar en bloque(seleccion)

alt + arrow up -> mover linea hacia arriba

alt + arrow down -> mover linea hacia abajo

control + click derecho  sobre un link -> va al archivo y si no existe lo crea en la ruta del link

control + click -> va a la definicion de la clase, funcion o variable sobre la que se realizo el click

control + alt + u -> to uppercase
control + alt + l -> to lowercase

control + d -> selecciona la proxima ocurrencia del string seleccionado en el archivo


control + shift + k -> borrar linea
control + shift + l -> select todas las ocurrencias


control + z -> deshacer
control + y -> rehacer


control + b -> mostrar/ocultar menu lateral(sitebar) 
control + k z ->  zen mode (modo cero distracciones)


control + shift + f -> find in folder

control + shift + p -> show all commands
